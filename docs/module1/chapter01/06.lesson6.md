# 软件架构的发展阶段

## 软件架构的过去

- 语言： C、Java、Python、Go
- 结构设计：数据结构、设计模式、算法
- 统一建模：UML、CBM

## 软件架构的现在

- 多视角：逻辑、物理、应用、技术、功能、部署
- 体系框架：ABSD、DSSA、AT

## 软件架构的未来

- 云化：IaaS、PaaS、SaaS资产复用
- 演进式架构：拆迁者，修缮者，绞杀者
