# 面试指导

## 赢取白富美

- 大型应用中架构设计的考量点
  - 高可用、吞吐量、扩展性、快速变更、成本平衡和安全性要素
- 职业生涯规划的问题
  - 目标不错，接下来2-3年如何付诸行动
  - 初期强调技术路线
  - 远期转管理路线、培养软实力
    - 技术调研
    - 产品能力
    - 大局观
    - 影响力的构建
    - 业务重要性 
- 你的什么特质使你从一个团队中脱颖而出
  - 阿里闻味官装B专用
  - 心理活动：全靠猪队友的衬托
